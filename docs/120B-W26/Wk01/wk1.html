Probability Spaces; Random Variables

High-Level Overview

  • Probability Space: \((\Omega, \F, \Prob)\) where:
    • \(\Omega\) denotes the outcome space
    • \(\F\) denotes the event space
    • \(\Prob: \F \to \R\) denotes the probability measure
  • A random variable \(X\) is a mapping \(X : \Omega \to \R\)
    • Support: \(X(\Omega)\) (set of values over which the random variable is defined)
      • Finite or countable support \(\implies\) \(X\) is discrete
      • Uncountable support \(\implies\) \(X\) is continuous

Probability Spaces; Random Variables

Example

Example 1

A fair coin is tossed twice, and the outcomes are recorded; let X denote the number of heads.

  • Outcome Space: \(\Omega =\) {H, T}2 = {(H, H), (H, T), (T, H), (T, T) }

  • X should map each element in \(\Omega\) to a real number, by counting the number of heads in the outcome.

    • X((H, H)) = 2;       X((H, T)) = 1;       X((H, T)) = 1;       X((T, T)) = 0

  • The support of X is {0, 1, 2}, which is finite, meaning X is a discrete random variable

  • Fair Coin implies we should use the equally-likely probability measure; \(\Prob(E) = |E| / |\Omega|\) for every event E.

    • Allows us to construct the probability mass function (PMF) of X.

Random Variables

Relevant Quantities

Discrete:

  • PMF: \(f(x) := \Prob(X = x)\)
  • Probabilities: \(\Prob(X \in B) = \sum_{x \in B} f(x)\)
  • CMF: \(F(x) := \Prob(X \leq x) = \sum_{t \leq x} f(x)\)
  • LOTUS: \(\E[g(X)] = \sum_{x} g(x) f(x)\)
  • Variance: \(\Var(X) := \E\left\{(X - \E[X])^2\right\}\)

Continuous:

  • PDF: \(f(x) \geq 0\) and \(\int_{\R} f(x) = 1\)
  • Probabilities: \(\Prob(X \in B) = \int_{B} f(x) \ \mathrm{d}x\)
  • CDF: \(F(x) := \Prob(X \leq x) = \int_{-\infty}^{x} f(t) \ \mathrm{d}t\)
  • LOTUS: \(\E[g(X)] = \int_{\R} g(x) f(x) \ \mathrm{d}x\)
  • Variance: \(\Var(X) := \E\left\{(X - \E[X])^2\right\}\)

Moment Generating Functions

Overview

Definition: MGF

The moment generating function (MGF) of a random variable X is defined as \[ M_X(t) := \E[e^{tX}] \]

Properties:

  • If it is continuous in a neighborhood of zero, then it uniquely determines a distribution.
  • \(M_X^{(n)}(0) = \E[X^n]\) for any \(n \in \N\) (Homework Hint: What is \(M_X(0)\)?)
  • \(M_{X + Y}(t) = M_X(t) \cdot M_Y(t)\) provided that \(X \perp Y\)
  • \(M_{aX + b}(t) = e^{tb} \cdot M_X(at)\) for constants \(a, b\)
